# AICoder 智能对话助手

AICoder是一款功能强大的智能对话助手应用，专为AI程序测试员和开发者设计，旨在提供高效、便捷的AI模型交互体验。

## 核心功能

- **多模型支持**：集成了DeepSeek、OpenAI、Anthropic Claude、Google Gemini等多种主流AI模型，满足不同场景的测试需求
- **多对话管理**：支持创建、删除和重命名对话，每个对话独立存储在`message_data`文件夹中
- **实时API用量统计**：防止API使用超量，让测试成本可控
- **Markdown格式支持**：包括数学公式处理，提升内容展示效果
- **文件上传功能**：方便分享代码和文档，支持多种文件格式
- **详细日志系统**：记录所有API调用和响应，便于测试分析
- **自定义API端点**：支持配置不同模型的API地址，灵活性高

## 技术特点

- **基于Tkinter的图形界面**：提供直观、友好的用户交互体验
- **多线程处理**：确保API调用不阻塞UI，提升应用响应速度
- **持久化存储**：对话历史和配置信息自动保存，支持跨会话恢复
- **路径适配**：支持本地开发和PyInstaller打包后的不同环境
- **错误处理**：完善的异常捕获和日志记录，提高应用稳定性

## 用户体验

- **窗口默认最大化**：提供更大的工作区域，便于查看长对话
- **快捷键支持**：按回车键发送消息，Shift+回车键换行，提高输入效率
- **主题切换**：支持深色/浅色主题，适应不同使用环境
- **响应式布局**：自动适应窗口大小变化，保持良好的视觉效果

## 系统要求

- 无需本地Python环境，已打包为可执行文件
- Windows 7+ 操作系统

## 安装与运行

### 方式一：直接使用打包好的可执行文件（推荐）

1. 下载我们提供的打包版本：
   - **OneFile单文件版本**：包含所有依赖的单个exe文件，方便携带和分发
   - **MultiFiles多文件版本**：包含主exe文件和多个依赖文件，启动速度更快

2. 解压下载的文件（如果是压缩包）
3. 双击运行 `AICoder.exe` 即可启动应用

### 方式二：从源码运行（开发人员）

1. 克隆或下载项目到本地
2. 安装依赖：`pip install -r requirements.txt`
3. 运行应用：`python index.py`

## 打包

使用PyInstaller打包为可执行文件：

### OneFile单文件版本
```bash
pyinstaller --onefile --windowed --add-data "aicoder.ico;" index.py
```

### MultiFiles多文件版本
```bash
pyinstaller --windowed --add-data "aicoder.ico;" index.py
```
